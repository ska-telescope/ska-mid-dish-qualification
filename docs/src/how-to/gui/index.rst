===================
DiSQ GUI user guide
===================

.. note::
    To install DiSQ, please follow the instructions on the 'Getting started' page.

DiSQ uses a TCP connection to communicate with an OPC-UA server implemented on a PLC, and, once connected, recursively scans the server for the required OPC-UA nodes.

This user guide describes how to connect the DiSQ GUI to a server and use the various functionality in the following sections:
